#INSTRUCTIONS

Run using Java version 8

##For merge sort run 
javac Main.java
java Main

##For Bloom Filter
Download BloomFilter.java, HashFunction.java, BloomFilterTest.java, positiveSet_clean.txt, and negativeSet_clean.txt.
CLI:
  javac BloomFilterTest.java
  java BloomFilterTest
  'runfiles'
  'exit'

Intellij:
  move *.java files to src folder
  move *.txt files to main directory (Intellij sets working directory as main directory by default)
  Run BloomFilterTest.java
  'runfiles'
  'exit'
