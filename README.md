
# CS361

A repository for CS361 coursework, including individual assignments and the group projects.

## Repository Structure

```
CS361/
├── src/
│   ├── assignments/
│       └── Homework1.java
│       └── Homework2.java
│       └── ...
│   ├── group-projects/
│       └── project1/
│           └── Main.java
│       └── project2/
│           └── Main.java
├── README.md
└── .gitignore
```

## Requirements

- Java JDK 8 (`1.8.0_471`)

## How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/CS361.git
   ```
2. Navigate to the project or assignment folder you want to run
3. Compile and run:
   ```bash
   javac FileName.java
   java FileName
   ```

## Branching Strategy

- `main` — stable code only
- `dev` — shared working branch
